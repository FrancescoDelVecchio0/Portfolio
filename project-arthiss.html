ArtHiss

Unity | Team Project | Generalist Programmer

Description

ArtHiss is a single player top-down game where the player controls a painter snake who must complete drawings on a canvas. By collecting paints, avoiding oil stains and obstacles, the snake grows longer and harder to control.

The gameplay combines classic Snake mechanics with creative painting and scoring combos, offering a unique blend of arcade challenge and artistic expression.

Download (Google Drive)

My Role

This was my first Unity project and my first time working within a large team. I was fully responsible for the programming some core systems:

Core UI

Integration UI and HUD elements, pause menu and selection levels.

Scoring

Built the combo-based scoring system and handled the integration of the HUD and UI elements.

Growth & Learning

Even as a first project, I managed to optimize the game to run smoothly. This experience was fundamental in learning how to structure complex Unity projects, collaborate with artists and designers, and make independent technical decisions under pressure.
